# 📝 CSV Quiz Application (Python)

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Type](https://img.shields.io/badge/Project-CLI-orange)
![Status](https://img.shields.io/badge/Status-Completed-success)

A Python-based Quiz Application that reads MCQ questions from a CSV file and evaluates user performance. The system calculates scores and displays final results in a simple CLI interface.

---

## 📂 Dataset
- Questions stored in CSV format  
- Each question includes options and correct answer  
- Optional database support for storing quiz data  

---

## ✨ Features

### 🔹 Quiz System
- Load questions from CSV file  
- Multiple-choice questions (MCQ)  
- Score calculation system  
- Final result display  

### 🔹 Data Handling
- CSV file-based question storage  
- (Optional) Database integration for quiz records  

### 🔹 User Experience
- Simple and interactive CLI interface  
- Easy navigation between questions  

---

## 🛠️ Tech Stack
- Python 🐍  
- CSV File Handling  
- SQLite (optional)  
- CLI (Command Line Interface)
