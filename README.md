# CSVquiz-python-project
Python CLI-based quiz system using CSV for input and database for storing quiz data and results.

📝 CSV Quiz Application

A command-line based Quiz Application built in Python that loads questions from a CSV file and stores quiz data in a database for efficient management.

✨ Features
Load quiz questions from CSV file
Multiple-choice questions
Score calculation and result display
Store quiz data in database
Simple and interactive CLI

🛠️ Tech Stack
Python
CSV File Handling
SQLite Database
CLI
